Kreeda Ankana 🏆⚽

Kreeda Ankana is a modern Android sports management application designed to simplify sports activity organization for players and teams. The app allows users to book sports slots, create and accept challenges, manage matches, and track scores/results in one centralized platform.

📌 Features


🔐 User Authentication (Sign Up / Login)

📅 Sports Slot Booking

⚔️ Challenge Board for finding opponents

🏟️ Ground & Match Management

🏆 Score Wall / Match Results

📖 Match History Tracking

🎨 Modern Sports-Themed UI

💾 Local Data Storage using RoomDB

📌 Problem Statement

Many local sports players and teams struggle with finding opponents, booking grounds, managing schedules, and tracking match activities efficiently. Kreeda Ankana solves this problem by providing a centralized and user-friendly sports management platform for organizing sports activities easily.


💾 Database

This project uses RoomDB for local offline data storage.

Data stored includes:

User information

Slot bookings

Challenges

Match results
